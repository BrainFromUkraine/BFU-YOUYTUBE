# Інструкція з встановлення

## Що потрібно

- Плата ESP32-C3 Super Mini
- Кабель USB-C
- Комп'ютер із встановленим [Thonny IDE](https://thonny.org/)
- Підключення до інтернету (для завантаження прошивки)

---

## Крок 1 — Прошивка MicroPython

1. Завантажте останню прошивку MicroPython для ESP32-C3:
   - Перейдіть на [https://micropython.org/download/esp32c3/](https://micropython.org/download/esp32c3/)
   - Завантажте останній файл `.bin` (наприклад, `ESP32_GENERIC_C3-20240602-v1.23.0.bin`)

2. Підключіть ESP32-C3 до комп'ютера через USB-C.

3. Відкрийте **Thonny IDE**.

4. Перейдіть до **Tools → Options → Interpreter**.

5. Виберіть **MicroPython (ESP32)** зі списку.

6. Виберіть правильний COM-порт (наприклад, `COM3` на Windows, `/dev/ttyUSB0` на Linux/macOS).

7. Натисніть **Install or update MicroPython (esptool)**.

8. У діалоговому вікні:
   - Виберіть порт пристрою
   - Натисніть **Browse** і виберіть завантажений файл `.bin`
   - Натисніть **Install**

9. Зачекайте завершення прошивки. Пристрій перезавантажиться автоматично.

---

## Крок 2 — Відкрийте файл проєкту

1. У Thonny натисніть **File → Open**.
2. Перейдіть до папки проєкту та відкрийте `src/main.py`.
3. Перед завантаженням вставте ваш YouTube API-ключ:
   ```python
   YOUTUBE_API_KEY = "PASTE_YOUR_API_KEY_HERE"
   ```
   Замініть заглушку на ваш реальний ключ.
   Дивіться [youtube-api.md](youtube-api.md) для інструкцій з отримання ключа.

---

## Крок 3 — Завантажте на пристрій

1. У Thonny натисніть **File → Save as…**
2. З'явиться діалог — виберіть **MicroPython device** (пристрій MicroPython).
3. Введіть ім'я файлу `main.py` і натисніть **OK**.
4. Файл збережено безпосередньо у флеш-пам'ять ESP32-C3.

---

## Крок 4 — Запустіть проєкт

1. Натисніть кнопку **Reset** на ESP32-C3 або зелену кнопку **Run** у Thonny.
2. Дисплей повинен засвітитися і показати головне меню.
3. Якщо дисплей залишається чорним — перевірте підключення, дивіться [wiring.md](wiring.md).

---

## Примітки

- Файл обов'язково повинен бути збережений як `main.py` на пристрої. MicroPython автоматично запускає `main.py` при увімкненні.
- Після завантаження кабель USB-C не потрібен. Пристрій працює автономно від будь-якого джерела живлення 3.3–5 В.
- Якщо потрібно повторно завантажити файл — підключіть USB і повторіть Крок 3.
